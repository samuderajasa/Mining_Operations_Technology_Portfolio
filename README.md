# Overview
This repository showcases selected work projects from my role as an Operations Technology Engineer, supporting mining operations through data analytics, reporting, and operational systems. The role involves developing Power BI dashboards and SQL-based data models to monitor production and operational performance using data sourced from Microsoft SQL Server.

The role also supports daily production monitoring and dispatch using Fleet Management Systems (FMS), ensuring effective resource deployment and alignment with daily, weekly, and monthly production targets while maintaining safety and productivity standards.
